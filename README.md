# DigitalEdu Chatbot Loader

This repository stores a fixed loader URL and a runtime JSON file.

- Fixed loader URL: https://cdn.jsdelivr.net/gh/enoz3/digitaledu-chat-loader@main/loader.js
- Runtime file (raw): https://raw.githubusercontent.com/enoz3/digitaledu-chat-loader/main/runtime.json
- Runtime file (mirror): https://cdn.jsdelivr.net/gh/enoz3/digitaledu-chat-loader@main/runtime.json
- Optional fallback: set secondary_base_url in runtime.json or data-secondary-base-url on script tag
